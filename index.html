Covid countries data visualization.

This page allows to select any country a get a chart for its COVID-19 confirmed dairy new cases or deaths. Data is taken from the European Centre for Disease Prevention and Control. Data is not loaded initially, press the load button to reload and update data at any moment. To show a graph, click on any country deaths or cases number in the table. You can have any number of simultaneous graphs, and close any one. Graphs are resizable (on some desktop browsers).

All source code is available here: https://github.com/carlos-urena/covid-countries-visu

This is work in progress.
